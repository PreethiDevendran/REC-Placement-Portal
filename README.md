# REC Placement Portal

## 📌 Overview

REC Placement Portal is a full-stack web application designed to bring transparency and accessibility to the campus placement process. The platform serves as a centralized hub where students can view placement drives, company information, eligibility criteria, placement statistics, and experiences shared by placed students.

The goal of this project is to bridge the information gap between placement coordinators, placed students, and juniors by providing accurate and up-to-date placement-related information.

---

## 🚀 Features

### Authentication

* Student Login
* New User Registration
* Secure User Management

### Dashboard

* View ongoing placement drives
* View previous placement drives
* Placement updates and announcements

### Companies Visited

* List of companies that have visited campus
* Company logos and details
* Number of students placed in each company

### Placement Statistics

* Year-wise placement statistics
* Total companies visited
* Placement trends and analytics

### Student Experiences

* Experience sharing platform
* Interview rounds and preparation tips
* Guidance from placed students

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/your-username/REC-Placement-Portal.git
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

Backend runs on:

```text
http://localhost:8080
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```text
http://localhost:3000
```

## 🎯 Problem Statement

In many colleges, placement-related information is often accessible only to students who actively participate in placement activities. Junior students usually have limited visibility into:

* Companies visiting the campus
* Eligibility requirements
* Placement statistics
* Recruitment processes
* Skills expected by companies

This project addresses these challenges by providing a transparent and centralized placement information system.
